> هذا هو **مصدر الحقيقة المختصر** لكل عمل واجهة مستقبلي. عند التعارض، هذه الوثيقة تحسم.
> للتفاصيل: `VISUAL_LANGUAGE.md` · `DESIGN_SYSTEM_RULES.md` · `COMPONENT_PATTERNS.md` · `MOTION_GUIDELINES.md` · `BRAND_DIRECTION.md` · `DESIGN_INSPIRATION.md`.

## 🧭 المبدأ الأعلى
> **«دفء الرصافة + انضباط Vercel»** — لوحة هادئة، لمسات توقيع، تايبوغرافي راقٍ، حركة هادفة.
> الشاشات الجديدة **تتبع هذه الوثائق**، ولا تخترع أنماطاً جديدة.

## ✅ افعل دائماً (Always)

1. استهلك التوكنز (`var(--f-*)`) — لا قيم ثابتة.
2. لوحة دافئة: `#fafaf7` نهاراً، `#131720` ليلاً.
3. الكبسولة (`999px`) لكل ما يُنقر؛ المستطيل المُدوّر للحاويات.
4. حدّ رفيع + ظل ناعم مزدوج + highlight داخلي علوي.
5. لوّن حسب القسم بنسبة مزج صغيرة (≤22%).
6. RTL أصيل (`inset-inline`, `margin-inline`).
7. خط Mono للأكواد/الأوقات/التسميات التقنية؛ Alexandria للعناوين.
8. أرقام جدولية (tabular-nums) في كل مكان.
9. حركة بـ`--f-ease`/`--f-spring` فقط + مسار `prefers-reduced-motion`.
10. صمّم الحالات الأربع: فارغ، تحميل، خطأ، حيّ.
11. أهداف لمس ≥44px + `touch-action: manipulation` + حلقة تركيز مزدوجة.
12. لون يحمل معنى: أخضر نجاح، عنّاب/أحمر خطر، ذهب تميّز.
13. كل حركة لها سبب (تسلسل/سرد/تغذية راجعة/انتقال حالة).
14. أظهر الحالة بصدق (حفظ تلقائي، تقدّم، تراجع، حدود الإجراء).
15. أيقونات الكهرباء المخصّصة حيث توجد بديل في `ds_icons.js`.

## ⛔ لا تفعل أبداً (Never)

1. أبيض نقي `#fff` كخلفية صفحة، أو أسود OLED `#000` ليلاً.
2. قيم لونية/قياسية ثابتة بدل التوكنز.
3. em-dash (—) في أي نص ظاهر — استخدم `-` أو فاصلة/نقطة.
4. لون قسم كتعبئة كاملة صاخبة.
5. ظلال صلبة أحادية الطبقة أو حدود سميكة.
6. easing عشوائي أو تحريك `width/height/top/left`.
7. حركة بلا سبب أو حلقة لانهائية بلا داعٍ.
8. `left/right` صريحة في تخطيط المحتوى (يكسر RTL).
9. أيقونات ستوك عامة حيث يوجد بديل برَندي مرسوم.
10. ادّعاء يقين لا تملكه (أجور نهائية قبل الكشف، إجراء بلا موافقة مطلوبة).
11. إغراق المستخدم بكل التعقيد دفعة واحدة (اكشف تدريجياً).
12. نسخ لقطة شاشة بعينها بكسل ببكسل — استلهم البنية والمبدأ.

## 🧩 أنماط يجب الحفاظ عليها (Preserve)
- لوحة الأقسام المتوسّعة (Deck `flex 1→2.7`).
- بطاقة Spotlight الداكنة المميّزة.
- البحث المتحوّل + زر الوضع الليلي MORPH.
- نافذة التراجع (Undo) 5 ثوانٍ.
- الستيكر ذاتي الرسم + المصادقة بالبريق.
- حلقة الطاقة (energy ring) + العلامة المائية للكود.
- التذييل التوقيعي (Playfair + معيّن ذهبي).
- نقطة الحفظ التلقائي المتنفّسة + علامة الصح المرسومة.

## 🚫 أنماط يجب تجنّبها (Avoid)
- بطاقات بيضاء متطابقة بلا تمييز (slop).
- تكرار نفس عائلة التخطيط في كل قسم.
- جداول بحدود تحت كل صف (استخدم تجميعاً/بطاقات).
- تباين منخفض في الوضعين.
- أيقونات بأطوال خطوط/عائلات مختلطة.

---

## 📏 معيار «تمّ» (Definition of Done)
شاشة جديدة لا تُسلَّم إلا بعد:
- [ ] توكنز لا قيم ثابتة.
- [ ] تباين AA (نص/أزرار/نماذج).
- [ ] RTL سليم.
- [ ] الوضعان (نهاري/ليلي) مُختبَران.
- [ ] `prefers-reduced-motion` مطبّق.
- [ ] أهداف لمس ≥44px.
- [ ] حالات: فارغ/تحميل/خطأ/حيّ.
- [ ] صفر em-dash.
- [ ] لون قسم بنسبة مزج صغيرة.
- [ ] الحركة مبرَّرة ومتّسقة فيزيائياً.
