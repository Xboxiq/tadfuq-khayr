# اللغة البصرية والتوكنز — تدفّق الخير

> مصدر الحقيقة لجميع التوكنز هو `:root` في `final.css`. هذه الوثيقة توثّقها وتشرح نيّتها.
> أي عمل تصميمي جديد **يجب** أن يستهلك هذه المتغيّرات، لا أن يُدخل قيماً ثابتة (hard-coded).

---

## 1. الألوان (Color)

### اللوحة الهادئة — الوضع النهاري
| التوكن | القيمة | الاستخدام |
|--------|--------|-----------|
| `--f-bg` | `#fafaf7` | الخلفية — أبيض دافئ مُصفرّ، **ليس** أبيض نقي |
| `--f-bg-2` | `#f4f3ee` | خلفية ثانوية / منخفضات |
| `--f-surface` | `#ffffff` | سطح البطاقات |
| `--f-surface-2` | `#f7f6f1` | سطح عند hover / تذييلات |
| `--f-surface-3` | `#efedde` | سطح ثالث / حالات مُعطّلة |
| `--f-border` | `#e7e5dc` | الحدّ الافتراضي (رفيع) |
| `--f-border-2` | `#d6d2c5` | الحدّ عند hover / فواصل أوضح |
| `--f-ink` | `#161513` | النص الأساسي — أسود دافئ مائل للبني |
| `--f-ink-2` | `#4a4940` | نص ثانوي |
| `--f-ink-3` | `#8a8478` | نص خافت / تسميات |

### لمسات التوقيع (Signature accents) — تُستخدم بجرعات صغيرة
| التوكن | القيمة | المعنى |
|--------|--------|--------|
| `--f-navy` | `#1e2a4d` | اللون الأساسي (primary) — الثقة والرسمية |
| `--f-navy-soft` | `#28366e` | نسخة أنعم |
| `--f-gold` | `#b8861b` | التميّز، التوقيع، «الخير» |

| `--f-crimson-soft` | `#c54a55` | نسخة أنعم |

### الحالة (Status) — دلالية فقط
| التوكن | القيمة | المعنى |
|--------|--------|--------|
| `--f-ok` | `#3a7a55` | نجاح / مكتمل |
| `--f-warn` | `#b87918` | تحذير / بانتظار |
| `--f-err` | `#b03a3a` | خطأ / خطر |

### لوحة الأقسام (Department palette) — OKLCH موحّدة الإضاءة
> القاعدة الذهبية: **إضاءة موحّدة (~0.52–0.57)، كروما مضبوطة** → ألوان «حيّة لكن غير فلوريسنت»، وأي قسم جديد ينتمي تلقائياً.

| التوكن | القيمة | القسم |
|--------|--------|-------|
| `--f-cs` | `oklch(0.52 0.105 259)` | الاشتراكات — كوبالت |
| `--f-ct` | `oklch(0.57 0.090 72)` | الفنية — برونز كراميل |
| `--f-cb` | `oklch(0.56 0.085 192)` | الفواتير — تركواز بحري |
| `--f-ca` | `oklch(0.54 0.095 12)` | الشكاوى — خشب وردي |

### الوضع الليلي — كحلي دافئ، **never OLED black**
| التوكن | القيمة | ملاحظة |
|--------|--------|--------|
| `--f-bg` | `#131720` | فحمي كحلي دافئ |
| `--f-surface` | `#1e2330` | سطح مرتفع بوضوح |
| `--f-ink` | `#f1efe6` | نص فاتح دافئ |
| ألوان الأقسام | `oklch(0.78–0.80 …)` | تُرفع إضاءتها للتباين على الداكن |

**قواعد اللون الصارمة:**
- لا تستخدم لوناً ثابتاً (hex) مباشرة — استهلك التوكن دائماً.
- مزج الألوان عبر `color-mix(in srgb, …)` لا بقيم يدوية.
- لون القسم يظهر دائماً بنسبة مزج صغيرة (8%–22%) على سطح، لا كتعبئة كاملة صاخبة.
- التباين: نص أساسي على سطح يحقق AA على الأقل (4.5:1).

## 2. التايبوغرافي (Typography)

| التوكن | الخطوط | الاستخدام |
|--------|--------|-----------|
| `--f-sans` | Geist → IBM Plex Sans Arabic | الأرقام، الواجهة اللاتينية، العلامة |
| `--f-arab` | Readex Pro → IBM Plex Sans Arabic → Geist | **نص الجسم العربي (الافتراضي للـ body)** |
| `--f-disp` | Alexandria → Readex Pro | العناوين والعرض (display) |
| `--f-mono` | Geist Mono → SF Mono | الأكواد، الأوقات، التسميات الصغيرة، الأرقام التقنية |
| Playfair Display | — | **حصراً** في التذييل التوقيعي (credit) |

**قواعد:**
- تتبّع حروف سالب على العرض: `-0.015em` إلى `-0.025em`.
- `letter-spacing: -0.005em` على الجسم العام.
- أرقام جدولية: `font-variant-numeric: tabular-nums` على كل الأرقام المتغيّرة (الرسوم، KPI).
- `font-feature-settings: "ss01", "cv01"` مفعّلة على الجسم.
- مقاسات العناوين سائلة: `clamp(1.7rem, 2.6vw, 2.2rem)` للعنوان الرئيسي.
- خط Mono دائماً بتتبّع حروف **موجب** (`0.04em`–`0.18em`) وحجم صغير — يوازن دفء العربية بطابع تقني.

## 4. الظلال (Shadows) — ناعمة، طبقية، أبداً ثقيلة

| التوكن | القيمة |
|--------|--------|
| `--f-shadow-xs` | `0 1px 2px rgba(22,21,19,0.04)` |
| `--f-shadow-sm` | تماس `1px` + highlight أبيض داخلي علوي |
| `--f-shadow` | تماس `1px` + هالة ناعمة `0 12px 28px -16px` |
| `--f-shadow-md` | تماس `1px` + هالة أوسع `0 24px 56px -20px` |

**القاعدة:** كل ظل = **جزء تماس قريب** (حاد، صغير) + **هالة بعيدة ناعمة** (انتشار سالب كبير). الوضع الليلي يستخدم هالة أعمق. لا ظلال صلبة أحادية الطبقة.

## 6. الشبكة والتباعد (Grid & Spacing)

- العرض الأقصى للمحتوى: `1480px` مركزي.
- الحشو الأفقي للصفحة: `24px`؛ العمودي: `32px` أعلى / `60px` أسفل.
- الفجوة بين أقسام الصفحة: `26px`؛ بين البطاقات: `12px`–`16px`.
- الشبكات سريعة الاستجابة:
  - الإحصاءات: 4 أعمدة ← 2 (≤1000px).
  - الأقسام: 4 ← 2 (≤1100px) ← 1 (≤600px).
  - العمودان (نشاط + جانبي): `1.55fr / 1fr` ← عمود واحد (≤1100px).
- استخدم الخصائص المنطقية دائماً (`inset-inline`, `margin-inline`, `border-inline-start`) لدعم RTL.
