# Design System v3 — تدفّق الخير

> **الاسم الرمزي:** Luminous Calm (هدوء مُضيء).
> طبقة تصميم **تطويرية ومعزولة تماماً** لا تمسّ التصميم الأساسي (`final*.css`).
> namespace خاص بها `--v3-*` و`.v3` فقط، فلا تعارض مع `--f-*` الأصلية.

---

## ما الجديد في v3 مقابل الأصل

| المحور | الأصل (v-final) | v3 (تطوير) |
|--------|-----------------|------------|
| **اللون** | توكنز مفردة (`--f-navy`...) | سلالم OKLCH كاملة (50→700) لكل لون توقيع، أسهل في التدرّج والحالات |
| **الأقسام** | 4 أقسام (CS/CT/CB/CA) | 5 أقسام (+`--v3-ce` طوارئ أرجواني) بنفس انضباط الإضاءة |
| **الظلال** | 4 مستويات | 5 مستويات (e0→e4) بطبقة تماس + هالة + إضاءة داخلية |
| **التايبوغرافي** | مقاسات ثابتة | سلّم سائل (clamp) بنسبة 1.20 |
| **الحركة** | easing + spring | + `ease-out` و`spring-soft` (تجاوز لطيف) و5 مدد |
| **المادّة** | زجاج علوي | + توكنز زجاج موحّدة + **الكرة المادّية اللامعة** كعنصر توقيع |
| **الحالات** | متفرّقة | مجموعة موحّدة: فارغ/تحميل(skeleton)/خطأ/مباشر/تراجع |

### العنصر التوقيعي الجديد: الكرة المادّية `.v3-orb`
مستلهَم من صورة التغذية البصرية #001 (Choose your plan)، لكنه **مُكيَّف لا منسوخ**:
- خامد = رمادي محايد دافئ؛ محدّد = الكحلي أو لون القسم بمزج مضبوط ضمن OKLCH (لا أزرق فاقع).
- لمعان علوي + ظل داخلي سفلي + هالة لونية = إحساس ثلاثي الأبعاد بضوء من الأعلى.
- يحترم `prefers-reduced-motion` ويعمل RTL.

## البنية


## الاستخدام

```html
<link rel="stylesheet" href="tokens.css">
<link rel="stylesheet" href="motion.css">
<link rel="stylesheet" href="components.css">

<body class="v3 v3-atmos">          <!-- أضف v3-dark للوضع الليلي -->
  <button class="v3-btn v3-btn--primary">ابدأ خدمة</button>
  <span class="v3-orb v3-orb--on"></span>
</body>
```

- **الوضع الليلي:** أضِف الصنف `v3-dark` على `<body>` (انظر زر التبديل في showcase).
- **الأقسام:** استخدم `--v3-cs/ct/cb/ca/ce` أو أصناف `v3-dept--cs` إلخ.

## الالتزام بقواعد steering (Design DNA)

| القاعدة | التطبيق في v3 |
|---------|----------------|
| لوحة دافئة | `#faf9f5` نهاراً، `#11151d` ليلاً (لا أبيض/أسود نقي) |
| توكنز لا قيم ثابتة | كل شيء عبر `var(--v3-*)` |
| الكبسولة للنقر | أزرار/شارات/مجزّأ = `--v3-r-pill` |
| RTL أصيل | `inset-inline`, `direction: rtl`, sheen بـtransform |
| لون القسم بمزج صغير | `color-mix … 12-22%` |
| حركة بـ ease/spring فقط | + reduced-motion إلزامي |
| صفر em-dash | لا em-dash في أي نص ظاهر |
| a11y | لمس ≥44px، `:focus-visible` ring، تباين AA |
| الحالات الأربع | empty/loading/error/live + undo |

## السكلز المستخدمة كقيود
`design-taste-frontend` (منع slop، أقفال اللون/الشكل) · `brandkit` (هوية ثابتة) ·
`animate` (لغة فيزيائية موحّدة) · `progressive-disclosure` (كشف تدريجي) ·
`mixed-initiative-flow` (المستخدم يقود الاختيار، النظام يحفظ تلقائياً) ·
`trust-calibration` + `transparency-patterns` (حالة مباشرة، حفظ تلقائي ظاهر) ·
`output-quality-rubrics` (معيار «تمّ»).

## تحديث v3.1 — تطبيق كامل للسكلز + التغذية البصرية

أضيفت طبقة `effects.css` وشاشة لوحة تحكّم مربوطة ببيانات حقيقية، مع تطبيق فعلي للسكلز الثمانية ومواقع التغذية البصرية.

### ملفات جديدة
```
effects.css        # grainy gradients (grainient) + bento utilities + Clash Display (fontshare)
dashboard.html     # لوحة تحكّم كاملة مربوطة بـ data.js (bento + كل المكوّنات الجديدة)
QUALITY_RUBRIC.md  # معيار جودة قابل للقياس + قائمة فحص (output-quality-rubrics)
```

### مكوّنات جديدة (مدفوعة بالسكلز)
- **شفافية/ثقة:** `v3-trust` (مؤكّد/تقديري/قيد التحديد)، `v3-pending-value` («تُحدّد بعد الكشف»)، `v3-gate` («يتطلب موافقة مدير»)، `v3-sync`، `v3-source`.
- **كشف تدريجي:** `v3-search` (بحث متحوّل)، `v3-disclose` (أقسام تتوسّع)، `v3-more`.
- **مبادرة مختلطة:** `v3-autosave` (النظام يقود)، `v3-confirm` (المستخدم يقود)، `v3-handoff`.

### التغذية البصرية المطبّقة
| الموقع | التطبيق الفعلي |
|--------|----------------|
| grainient.supply | `.v3-grain` + `.v3-grad-*` (تدرّجات بحبيبات فيلمية) |
| bentogrids.com | `.v3-bento` (شبكة 12 عمود غير متماثلة) |
| fontshare.com | Clash Display لأرقام الـhero (`.v3-hero-num`) |
| curated.design | تركيب تحريري متحفّظ + تسلسل |
| صورة #001 | `.v3-orb` (الكرة المادّية) |

> راجع `QUALITY_RUBRIC.md` لجدول تتبّع السكلز الثمانية ومواقع التغذية البصرية مقابل أماكن تطبيقها.
